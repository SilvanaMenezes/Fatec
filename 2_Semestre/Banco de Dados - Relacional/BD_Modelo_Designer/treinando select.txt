Selecionar os nomes que começam com a Letra J
SELECT nome_pessoa FROM pessoa WHERE nome_pessoa LIKE 'J%';

Selecionar os nomes que terminam com a letra A
SELECT nome_pessoa FROM pessoa WHERE nome_pessoa LIKE '%A';
SELECT nome_pessoa FROM pessoa WHERE nome_pessoa NOT LIKE '%A';

Selecionar os nomes que tem A
SELECT nome_pessoa FROM pessoa WHERE nome_pessoa LIKE '__A';

Mostre o Endereço e  o Nome do Proprietario das unidades Condominais

SELECT unidade_condominial.id_unid_cond, unidade_condominial.endereco_unid_cond, pessoa.nome_pessoa FROM unidade_condominial INNER JOIN proprietario ON unidade_condominial.id_unid_cond = proprietario.Unidade_Condominial_id_unid_cond INNER JOIN pessoa on pessoa.id_pessoa = proprietario.Pessoa_id_pessoa ORDER BY id_unid_cond ASC;
